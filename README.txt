Требование к ОС: Ubuntu

Сборка:
1. Запустить терминал
2. Перейти в папку Parser
3. Запустить скрипт build, введя ./build

Запуск:
1. Запустить терминал
2. Перейти в папку Parser
3. Ввести в терминале ./Parser путь_к_файлу, например ./Parser ./tests/KeyWords.txt

Тесты:
Тесты расположены в папке tests
